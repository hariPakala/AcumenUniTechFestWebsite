Code Enigma:
No. of rounds: 2
Team size: 2
Time Limit:

  • 1st round : 30 minutes
  • 2nd round : 2 hours

Description:
Round 1:

  • This competition will be on paper.
  • A set of programs will be given in two sections.
  • Section 1: A set of 5 difficult-to-understand short programs will be given (in C,C++ or Java). The output of each program has to be predicted and written.
  • Section 2: Another set of 5 difficult-to-understand short programs are given (in C, C++ or Java). The output of each program has to be predicted. This predicted output will be a sentence in jumbled form. The correct sentence has to be formed and written.
  • The teams who have scored more than the cut-off will be qualified for the next round.

Round 2:

  • This competition will be on computer systems.
  • A set of 5 very difficult problems will be given.
  • Each problem explains a scenario and describes a problem statement based on the scenario.
  • A program has to be written to solve the problem stated.
  • The program must satisfy the input and output formats specified in the problem.

It should be written in C, C++ or Java and must not include any methods, etc. except stdin and stdout (standard input-output methods).

  • The program should be compiled, executed and saved on the computer system itself.
  • Teams will be ranked according to the number of problems solved accurately. Partially solutions to the problems will also be considered accordingly.

If there is any tie, the team with greater code efficiency and earlier submission time will be given higher priority.
